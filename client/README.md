# workout-tracker-mini client

This folder contains the React frontend for `workout-tracker-mini`.

## Responsibilities

- login UI (guest + user)
- workout CRUD interface
- custom exercise management
- accessibility preferences UI (high contrast + text size)

## Local Development

From repo root:

```sh
pnpm run dev
```

Vite runs on `http://localhost:5173` and proxies `/api` calls to the backend server.
